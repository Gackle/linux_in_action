# fg 命令

`fg` 命令用于将后台作业（在后台运行的或者在后台挂起的作业）放到前台终端运行。与 `bg` 命令一样，若后台任务中只有一个，则使用该命令时，可以省略任务号。

## 命令用法

``` shell
fg [job_spec]
```

其中，`job_spec` 作业号可以通过 `jobs` 命令查找。

# bg 命令

`bg` 命令用于将作业放到后台运行，使前台可以执行其他任务。该命令的运行效果与在指令后面添加符号 `&` 的效果是相同的，都是将其放到系统后台执行。

## 命令用法

``` shell
bg [job_spec ...]
```

其中，`job_spec` 作业号可以通过 `jobs` 命令查找。
